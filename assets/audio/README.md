# Music

Three tracks, referenced by `scripts/audio/audio_controller.gd`:

| File | When it plays |
|------|---------------|
| `music_normal.ogg` | Walking the ship, nothing critical |
| `music_panic.ogg`  | A CRITICAL fault is active |
| `music_stasis.ogg` | In the pod |

These are the only sounds in the game that are not synthesised — everything short and
mechanical is generated by `scripts/audio/sound_forge.gd`.

**They are allowed to be missing.** `AudioController` checks with `ResourceLoader.exists()`,
logs once and plays silence, so the game runs and the tests pass long before the music
lands. Drop the files in and they start working with no code change.

**Loop points matter more than the composition.** Each track loops for minutes at a time, so
export them with clean loop boundaries — in Godot, tick *Loop* on the `.ogg`'s import
settings once the file is here.
